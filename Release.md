# React native RELEASE
Step 1. change data.js version   
Step 2. Run config  
Step 3. `git commit`  
  
### Android

 run Android:release  
  
### IOS

2. Check bundle id
3. Click product > Archive
4. Auto pop up Organise(or click organize) and enter ACCESS pwd 
5. next to upload
6. 等一段時間確認上傳成功
7. go to test flight tab check success
